* Overview
This package provides comprehensive Salesforce development tools for Emacs, including:

- Apex development support
- SOQL query execution
- Salesforce DX integration
- Lightning Web Components (LWC) support
- Metadata deployment and retrieval
- Test execution and debugging
- Data import/export tools

* Dependencies
+ [[https://developer.salesforce.com/docs/atlas.en-us.sfdx_setup.meta/sfdx_setup/sfdx_setup_install_cli.htm#sfdx_setup_install_cli_npm][Salesforce CLI]]
+ Emacs 27.1 or later
+ treesit (for syntax highlighting)

* Installation
** Doom Emacs
#+begin_src emacs-lisp
(use-package! salesforce-minor-mode
   :recipe (:host github :repo "furucrm-minhtan/salesforce-minor-mode" :files ("*.el")))
#+end_src

** Vanilla Emacs
Clone the repository and add to your load path:
#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/salesforce-minor-mode")
(require 'salesforce-minor-mode)
