#+title: Salesforce Development Tools for Emacs

* Overview
A comprehensive Emacs package for Salesforce development, integrating Salesforce CLI with Emacs for a streamlined development workflow.

** Key Features
- Full Salesforce CLI integration
- Syntax highlighting with tree-sitter (Apex, SOQL, Visualforce, LWC)
- LSP support with Eglot/LSP-bridge
- Org-babel support for literate programming
- Multi-org management
- Data import/export with org-mode integration
- Consult integration for unified search

* Installation
** Dependencies
- [[https://developer.salesforce.com/docs/atlas.en-us.sfdx_setup.meta/sfdx_setup/sfdx_setup_install_cli.htm][Salesforce CLI]] (sf) v2.x+
- Emacs 27.1+
- [[https://tree-sitter.github.io/tree-sitter/][tree-sitter]]

** Doom Emacs
#+begin_src emacs-lisp
;; packages.el
(package! salesforce-minor-mode
  :recipe (:host github 
           :repo "furucrm-minhtan/salesforce-minor-mode" 
           :files ("*.el" "*/*.el")))

;; config.el
(use-package! salesforce-minor-mode
  :hook ((apex-ts-mode . salesforce-mode)
         (soql-ts-mode . salesforce-mode)
         (visualforce-ts-mode . salesforce-mode)
         (lwc-ts-mode . salesforce-mode)))
#+end_src

** Vanilla Emacs
#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/salesforce-minor-mode")
(require 'salesforce-minor-mode)

(add-hook 'apex-ts-mode-hook #'salesforce-mode)

** Key Bindings
*** Main Prefix Keys
| Key   | Action                   |
|-------+--------------------------|
| =M-o o= | Org management menu      |
| =M-o r= | Run/Execute code menu    |
| =M-o R= | Resource management menu |
| =M-o A= | Authorize org            |

*** Org Management (=M-o o=)
| Key | Action               |
|-----+----------------------|
| =TAB= | Switch org           |
| =.=   | Open org in browser  |
| =r=   | Retrieve metadata    |
| =d=   | Deploy/push metadata |
| =p=   | Diff local vs remote |

*** Run/Execute Code (=M-o r=)
| Key | Action              |
|-----+---------------------|
| =q=   | Execute SOQL query  |
| =s=   | Execute SOSL search |
| =a=   | Execute Apex code   |

*** Resource Management (=M-o R=)
| Key | Action                     |
|-----+----------------------------|
| =c=   | Create Salesforce resource |
| =l=   | Clear log data             |

* Features
** Apex Development
- Syntax highlighting and completion
- Execute code and run tests
- Generate classes and triggers
- Org-babel support:
 #+begin_src org
  ,#+begin_src apex :org myorg
    System.debug('Hello from Emacs!');
  ,#+end_src
 #+end_src

** SOQL Queries
- Query execution with results visualization
- Auto-completion for fields and objects
- Org-babel support:
  #+begin_src org
  ,#+begin_src soql :org myorg :results table
  SELECT Id, Name FROM Account LIMIT 10
  ,#+end_src
  #+end_src

** Data Operations
- Bulk import/export
- Org-table integration
- CSV and JSON support

** Multi-Org Management
- Authenticate and switch between orgs
- List and filter orgs (sandbox, production, scratch)
- View org status and logs

** Project Management
- Initialize projects
- Deploy/retrieve metadata
- Compare local and remote changes
- Ediff integration for conflicts
